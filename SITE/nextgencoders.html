What is NextGenCoders?

NextGenCoders is an organization made to help kids age 8 to 16 learn python. Learners recive 50 minutes of learning python and 10 minutes of having fun.
Details                                                     ▼

What we are going to learn on each day of class:

  • Your first python script!
  • Object Types
  • Variables
  • Input
  • The While Loop
  • The For Loop
  • The If Statment
  • Else and Elif
  • Lists
  • Other Data Structures
  • Making a Quiz Game!
  • Functions
  • The Return Statment
